Exercises on

Javascript

Check out my exercises on Javascript!

Exercise 1

Using the text editor Atom, the first exercise was to create a simple website page by manipulating simple Javascript codes. I learned how to code an HTML element that is being manipulated with JS, similar to a customized navigation bar. I also learned how to display text in an element using JS and how to add a point counter that increases or decreases a variable when elements are pressed using buttons.

Introduction to Javascript

Exercise 2

This second exercise is about learning how to use eventListeners, functions and conditional statements to create an example that asks for information and then changes text and an HTML element based on the information received. More precisely, the user will be able to type in their name at the very beginning to then access the page and a welcome message will be displayed. The user can interact with the welcome message and accordingly the background colour will change when a button is clicked. After a few seconds, a fortune generator will be displayed and the user can generate their fortune!

Your Daily Fortune with Event Listeners, Functions and Conditional Statements

Exercise 3

This third exercise is about learning how to use arrays and loops to create a set of buttons that have the same class and that have the ability to increase and decrease points. There are also a set of buttons that disappear when clicked and a pop-up message for when the points reach 100.

Arrays and Loops
